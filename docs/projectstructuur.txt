1. public/
Bevat statische bestanden die de browser direct kan benaderen. Denk aan de index.html waarin de React-app wordt geladen.

2. src/
Bevat de kern van je applicatie, waaronder:

components/: Alle herbruikbare React-componenten.
pages/: De hoofdweergaven die via React Router bereikbaar zijn.
context/: Voor het managen van globale state met React Context API.
hooks/: Aangepaste hooks voor data-ophaling en opslag.
services/: API-logica zoals het ophalen van prompts en interactie met OpenAI.
styles/: CSS-bestanden en Tailwind-configuraties.
data/: JSON-bestanden met alle prompts en vertalingen.
3. utils/
Handige hulpfuncties die overal in de app kunnen worden hergebruikt, zoals string-helpers of API-verzoeken.

4. App.jsx en main.jsx
App.jsx is het hoofdcomponent waarin routes en context worden geregeld.
main.jsx is het entry-point waarin React wordt gerenderd in de index.html.
5. .env
Om gevoelige gegevens zoals API-sleutels (voor OpenAI) buiten de codebase te houden.

6. package.json
Projectconfiguratie waarin alle dependencies (React, Axios, Tailwind, etc.) zijn vastgelegd.
